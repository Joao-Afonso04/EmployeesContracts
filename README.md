Desenvolvido um sistema completo de gestão de contratos de trabalho em Java, permitindo registrar departamentos, funcionários com diferentes níveis de senioridade e seus contratos horários.

🛠️ Ferramentas e Tecnologias Utilizadas:
Java - Linguagem principal
Orientação a Objetos - Estrutura com classes (Worker, HourContract, Departament, WorkerLevel)
Collections - ArrayList para armazenar contratos
Calendar API - Para processamento de datas e cálculo de income por período
Swing - Interface gráfica Desktop
✨ Funcionalidades:
✅ Criar trabalhadores com nível (Junior, Mid-Level, Senior)
✅ Registrar contratos horários por data
✅ Calcular income mensal baseado em contratos
✅ Interface gráfica intuitiva com validação de dados
✅ Tratamento de erros robusto

💡 Aprendizados:
Manipulação eficiente de datas em Java
Uso de enums para níveis de trabalhadores
Integração de interface gráfica com lógica de negócio
Validação e tratamento de exceções
A interface Swing foi desenvolvida com assistência de GitHub Copilot, facilitando a criação de uma UI responsiva e bem estruturada.
